Discover What's Inside

Orga is built with powerful, industry-standard technology for your peace of mind.

Dual-Mode Security Architecture

Choose the protection that fits your needs. Orga provides two distinct modes for flexibility and control.

  • Advanced Security Mode: Utilizes Apple's native CryptoKit framework for hardware-accelerated AES-26-GCM authenticated encryption for maximum protection.
  • Rapid Hide Mode: A high-speed 'stealth' mode that instantly obscures files, making them invisible on the system. It's the perfect choice for everyday privacy and convenience.

Secure Authentication System

Your data is protected by multi-layered authentication. Passwords are never stored.

  • Password Hashing: Uses a secure SHA-512 salt-and-hash system.
  • Keychain Integration: Securely stores the password hash, salt, and encryption keys in the macOS Keychain.
  • Biometric Login: Full integration with Touch ID using the LocalAuthentication framework.

Advanced Temporary File Management

Orga leaves no trace. Decrypted files are managed in a secure, self-destructing temporary cache.

  • Implemented a multi-layered security strategy for temporary decrypted files.
  • Automatically cleans up the entire temporary directory on every launch.
  • A 5-minute "self-destruct" timer automatically deletes temporary files when the app is inactive.

Modern, Minimalistic User Interface

A beautiful, responsive UI built with a professional sidebar layout using SwiftUI.

  • Category-based filtering system (All, Images, Videos, etc.).
  • Designed and implemented a central Theme.swift style guide for brand consistency.
  • Polished user experience with transparent UI elements and minimalistic effects.
